# Features
* Includes the [Euler gamma function](https://en.wikipedia.org/wiki/Gamma_function)
* Multiple pseudogamma functions, including:
  * [Hadamard's gamma function](https://en.wikipedia.org/wiki/Hadamard%27s_gamma_function)
  * [Luschny factorial](https://www.luschny.de/math/factorial/hadamard/HadamardsGammaFunction.html)
  * [Klimek's K-function](https://ar5iv.labs.arxiv.org/html/2107.11330)
  * [Ulgenes' lambda function](https://arxiv.org/pdf/2301.09699)
* Factorial (*n*!)
* Combinatorics functions (*n*C*r* & *n*P*r*)
* Factorial power functions (*x*<sup>(*n*)</sup> & (*x*)<sub>*n*</sub>)

# Notes
This isn't perfect because Ulgenes' lambda uses an infinite product for computation.
